# TI de Bastidores — Plano de Comunicação do DITI (Hortolândia)

Repositório para o **Plano de Marketing Institucional** e os **materiais de comunicação** do **Departamento de Infraestrutura de TI (DITI)**.

## Objetivos
- Dar visibilidade às entregas de infraestrutura (rede, data center, segurança, continuidade).
- Educar as secretarias sobre boas práticas (uso da rede, requisições, governança).
- Alinhar mensagens com o Planejamento Estratégico e com a LGPD.

## Entregáveis
- Apresentações (PPT/PDF) para gestores.
- Cartazes/avisos operacionais (rede, segurança, backup).
- Comunicados e notas técnicas padronizadas.
- Calendário de publicações (intranet, site).
